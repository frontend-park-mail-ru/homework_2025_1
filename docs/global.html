partition(array, predicate) → {Array.<Array>}

Разделяет массив на два массива в зависимости от предиката
Parameters:
Name Type Description
array Array Исходный массив
predicate function Функция предикат, возвращающая true или false
Source:

Returns:
Двумерный массив, где первый массив содержит элементы, удовлетворяющие предикату, а второй — нет
Type
Array.<Array>
Example
// returns [[2, 4, 6], [1, 3, 5]]
partition([1, 2, 3, 4, 5, 6], num => num % 2 === 0);
