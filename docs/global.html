deepMerge(obj1, obj2) → {object}

Глубоко объединяет два объекта. Если оба объекта содержат одинаковые ключи, и значения по этим ключам — объекты, то они объединяются рекурсивно. Если значения — не объекты (примитивы, массивы и т. д.), то значение из второго объекта перезаписывает значение из первого.
Parameters:
Name Type Description
obj1 object Первый объект.
obj2 object Второй объект.
